## 🎯 目標

「ド素人でも間違えない最強の入力インターフェース」を実装する。

### 成功定義

- ✅ システム経由受注率: 100%
- ✅ アンバサダー稼働率: 80%以上
- ✅ 受注ミス率: 0%

---

## 📋 実装スコープ

### 1. Visual Ordering画面（最優先）

#### 1.1 顧客情報入力画面

- 顧客情報フォーム
- バリデーション

#### 1.2 採寸入力画面

- ビジュアル人体図
- タップ入力
- 音声入力（将来対応）

#### 1.3 生地選択画面

- 画像グリッド表示
- 在庫ステータスバッジ表示
- フィルター・検索機能

#### 1.4 仕様選択画面

- ビジュアルオプション選択
- あり得ない組み合わせのブロック
- リアルタイム価格計算

#### 1.5 見積もり確認画面

- 注文内容確認
- 価格表示
- 送信ボタン

---

### 2. Home（Dashboard）画面

- KPIカード表示
- タスクリスト表示

---

### 3. Inventory（生地一覧）画面

- 生地一覧表示
- フィルター・検索
- 在庫確保機能

---

## 📐 画面フロー

```
Home (Dashboard)
  ├─ 新規注文作成 → Visual Ordering
  │                     ├─ 顧客情報入力
  │                     ├─ 採寸入力
  │                     ├─ 生地選択
  │                     ├─ 仕様選択
  │                     └─ 見積もり確認 → 注文送信
  └─ Inventory → 生地一覧 → 生地確保
```

## 🛠️ 技術スタック

- **フレームワーク**: Flutter 3.16.0+
- **状態管理**: Riverpod
- **オフライン**: Hive
- **画像キャッシュ**: cached_network_image
- **認証**: Firebase Auth
- **API通信**: http
