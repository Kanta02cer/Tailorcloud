# TailorCloud: LOI獲得戦略（シード調達前）

**作成日**: 2025-01  
**目標**: LOI（利用意向書）10件獲得  
**期間**: 1〜2週間

---

## 🎯 目標

**「これ月5,000円なら使う？」** と聞いて回り、**LOI（利用意向書）10件を集める**。

これが、J-KISSで3,000〜5,000万円の資金調達を成功させる鍵です。

---

## 📋 LOI獲得のプロセス

### ステップ1: デモ準備（1〜2日）

#### 1.1 システムの起動

```bash
# 1. システム状態確認
./scripts/check_system.sh

# 2. 環境変数セットアップ
./scripts/setup_local_environment.sh

# 3. バックエンドAPI起動（ターミナル1）
./scripts/start_backend.sh

# 4. Flutterアプリ起動（ターミナル2）
./scripts/start_flutter.sh
```

#### 1.2 デモデータの準備

- **顧客データ**: 3〜5名のサンプル顧客を登録
- **生地データ**: 3〜5種類のサンプル生地を登録
- **発注書生成**: 1〜2件のサンプル発注書を生成

#### 1.3 デモの練習

- **デモ時間**: 5分以内
- **デモ内容**:
  1. クイック発注画面の説明（30秒）
  2. 顧客選択・生地選択（1分）
  3. 金額・納期入力（1分）
  4. 発注書生成（30秒）
  5. PDF表示（30秒）
  6. 質問タイム（2分）

**練習回数**: 最低3回（家族・友人に見てもらう）

---

### ステップ2: ターゲットリストの作成（1日）

#### 2.1 優先ターゲット

**最優先（既存の知り合い）**:
- [ ] 知り合いのテーラー（5名）
- [ ] 知り合いの縫製工場（3名）
- [ ] 業界関係者（2名）

**優先（紹介経由）**:
- [ ] 知人の紹介で接触可能なテーラー（10名）
- [ ] 知人の紹介で接触可能な工場（5名）

**標準（直接アプローチ）**:
- [ ] 地域のテーラーショップ（20名）
- [ ] 地域の縫製工場（10名）

#### 2.2 連絡方法の優先順位

1. **直接訪問**（最優先）
   - 最も効果的
   - デモがしやすい
   - 信頼関係を築きやすい

2. **電話**（優先）
   - 直接訪問が難しい場合
   - アポイント取得

3. **メール**（標準）
   - 初回接触
   - フォローアップ

### ステップ3: アプローチ戦略

#### 3.1 オープニング（最初の30秒）

**テンプレート**:

> 「こんにちは。私はTailorCloudという、テーラー向けの業務システムを作っている者です。
> フリーランス保護法に対応した発注書を、スマホで3分で作れるシステムです。
> 5分だけお時間いただいて、デモを見ていただけませんか？」

**ポイント**:
- 短く簡潔に
- ベネフィットを明確に
- 時間のリクエスト（5分）

#### 3.2 デモの流れ（5分）

1. **問題提起**（30秒）
   - 「フリーランス保護法への対応、大変ですよね」
   - 「手作業で発注書を作るの、時間かかりますよね」

2. **デモ実行**（3分）
   - クイック発注画面を開く
   - 顧客選択 → 生地選択 → 金額入力 → 発注書生成
   - 「これで3分で発注書が完成します」

3. **価値提案**（1分）
   - 「フリーランス保護法に対応したPDFが自動生成されます」
   - 「これで法対応の手間がなくなります」

4. **質問タイム**（30秒）

#### 3.3 クロージング（最後の1分）

**質問**:

> 「このシステム、月5,000円なら使っていただけますか？」

**回答への対応**:

- **「使う」** → LOIを書いてもらう
- **「検討する」** → フォローアップを約束
- **「使わない」** → 理由を聞いて改善に活かす

```
【理由】
_______________________________________________
_______________________________________________

【希望開始時期】
2025年X月

【署名】

```

#### 4.2 LOI獲得のポイント

- **無料トライアル** を提示（最初の1ヶ月は無料）
- **早期導入割引** を提示（最初の10店舗は半額）
- **押し付けない**（断られたら感謝して次へ）
